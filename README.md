
# Python Project 1
## Learning Python with Poetry

### Домашнее задание 11.1 — генераторы

В этом проекте реализован модуль с генераторами для работы с последовательностями данных из домашнего задания 11.1.

Реализованные генераторы:
- Генератор №1: последовательно возвращает элементы входной последовательности, отфильтрованные по заданному условию.
- Генератор №2: преобразует элементы входной последовательности по заданной функции и возвращает результаты по одному.
- Генератор №3: объединяет несколько последовательностей и по очереди возвращает элементы из каждой.

Все генераторы оформлены в отдельном модуле в папке `src` и покрыты юнит‑тестами в папке `tests`.

### Тесты и покрытие

- Для проверки использован `pytest` вместе с плагином `pytest-cov`.
- HTML‑отчёт о покрытии кода расположен в папке `htmlcov`, основной файл отчёта — `htmlcov/index.html`.
- Отчёт добавлен в репозиторий для проверки наставником.
